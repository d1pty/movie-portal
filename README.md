Веб-приложение для просмотра фильмов и сериалов с возможностью навигации по жанрам, спискам и подробной информации о контенте.
## ⚙️ Установка и запуск

### Клонирование проекта

```bash
git clone https://github.com/d1pty/movie-portal.git
cd movie-portal
```

---

## Backend

```bash
cd backend
npm i
npm run
```
## Frontend

```bash
cd frontend
npm i
npm run
```
